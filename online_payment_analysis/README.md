# Онлайн төлемдерді бақылау және талдау платформасы

Бұл жоба Python тілінде жазылған оқу жобасы.  
Мақсаты — онлайн төлемдерді сақтау, талдау және алаяқтықты анықтау.

## Құрылымы
- `models/` — Төлем моделі (Payment)
- `analysis/` — Төлем анализаторы (FraudAnalyzer)
- `utils/` — Статистика класы (PaymentStats)
- `main.py` — Негізгі файл (жобаны іске қосу)

## Іске қосу
```bash
python main.py
```

## Пайдаланылған тақырыптар
1. Класс, инкапсуляция, аннотациялар  
2. Арнайы әдістер (__str__), property  
3. Мұрагерлік және полиморфизм (ABC)  
4. Әдістер (staticmethod, classmethod)  
5. Архитектура құрылымы  
6. Құжаттау және PEP 8 стилі
